P
Product API
Quickstart | API Definition | FAQ & Guides | Sample Messages
The Product API enables Expedia partners to read, create, and edit room types and rate plans via the API, without having to use ExpediaPartnerCentral (EPC) or contact their market manager. It also offers the possibility to read basic property configuration to better use the API to manage room types and rate plans afterwards.
M
Marketplace Feed
Quick Start | Marketplace Feed API | Try It
The Marketplace Feed API enables hoteliers to assess how competitive they are in the Expedia marketplace. This feed currently offers insights into how competitive a hotel's availability and rates are, as well as feedback provided by Expedia guests.
